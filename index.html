📝 Redação Bot 4.0

Automação inteligente para o Redação Parana

Python PyAutoGUI Tkinter

🚀 Sobre o projeto

O Redação Bot 4.2 é um aplicativo desenvolvido em Python que automatiza a escrita de redações em qualquer editor ou site, utilizando automação de teclado e mouse.

Esta versão marca a transição de um simples script para um software completo, com interface moderna, ferramentas auxiliares e execução segura da automação.

✨ Funcionalidades

  • * Interface gráfica moderna (Tkinter + CustomTkinter)
  • *Editor dedicado para redações
  • *Captura visual de coordenadas com mira central
  • *Automação com Alt+Tab, clique e digitação automática
  • *Painel de ferramentas flutuante
  • *Acesso rápido ao ChatGPT e QuillBot
  • *Execução inteligente (só libera quando tudo está pronto)
  • *Geração de executável (.exe)

🧠 Como funciona

  1. O usuário escreve a redação no aplicativo
  2. Define o ponto de escrita usando a mira de coordenadas
  3. O botão Executar Automação é liberado
  4. O bot troca de janela, clica no local e digita o texto

🛠️ Tecnologias utilizadas

  • Python 3.10+
  • Tkinter
  • CustomTkinter
  • PyAutoGUI
  • Webbrowser

📦 Instalação (modo desenvolvedor)

git clone https://github.com/seu-usuario/redacao-bot-4.0.git
cd redacao-bot-4.0
pip install pyautogui customtkinter
python redacao4.py

🧱 Gerar executável (.exe)

python -m pip install pyinstaller
python -m PyInstaller --onefile --windowed redacao4.py

⚠️ Avisos importantes

  • Antivírus podem acusar falso positivo devido à automação
  • Use com responsabilidade
  • Não utilize para burlar sistemas de avaliação
